为下列表达式构造 DAG，且指出他们每个子表达式的值编码。假定 + 是左结合的。

1. 1a+b+(a+b)

1)  id  a
2)  id  b
3)  +   1   2  // a + b
4)  +   3   3


2. a+b+a+b

1) id   a
2) id   b
3) +    1   2   // a + b
4) +    3   1   // a + b + a
5) +    4   2   // a + b + a + b


3. a+a+(a+a+a+(a+a+a+a))

1) id   a
2) +    1   1   // a + a
3) +    2   1   // a + a + a
4) +    3   1   // a + a + a + a
5) +    3   4   // a + a + a (a + a + a + a)
6) +    2   5
